Учебный класс

Алиф Академия совместно с ТИКА открыла учебный класс, куда может прийти каждый желающий у кого нет компьютера или доступа к интернету. Для этого вам необходимо быть участником одного из курсов Открытой Школы JS. Время посещения класса можно уточнить у куратора курса.

На данный момент класс функционирует по адресу: г. Душанбе, ТЦ "Баракат". Ориентир: школа № 8, 2 этаж.
